# 덕션 브랜드 북

> 작성일: 2026-07-11 · 디자인 시스템의 상위 문서 — 모든 카피·화면·마케팅은 여기서 출발한다.

---

## 1. 브랜드 핵심

### 이름
**덕션 (Duction)** — 덕질 + Auction.
읽는 순간 "덕후의 경매"라는 뜻이 통하고, 두 글자라 워드마크·앱 아이콘·해시태그(#덕션)에 강하다.

### 한 문장 정의
> 브라이스 수집가를 위한 **옥션 하우스**.

"중고장터"가 아니라 "옥션 하우스"다. 크리스티가 미술품을 다루는 격식으로
인형을 다루되, 그 격식을 커뮤니티의 다정한 말로 전달한다.

### 태그라인
**"아끼던 아이에게, 제값을 아는 새 집사를."**
- 판매자에게: 헐값에 보내지 않아도 된다 (경매 = 시장이 제값을 찾는다)
- 구매자에게: 당신은 '구매자'가 아니라 '새 집사'다 (커뮤니티 vernacular)
- 유틸리티 문맥용 짧은 버전: **"브라이스 옥션 하우스"**

### 브랜드가 지키는 약속 (= 기능의 브랜드 번역)
| 약속 | 뒷받침하는 기능 |
|---|---|
| 기록은 전부 공개합니다 | 입찰 로그 공개, 낙찰가 히스토리(시세) |
| 돈은 우리가 지킵니다 | 에스크로, 미결제 페널티 |
| 정품과 팩토리를 섞지 않습니다 | 강제 구분 표기 + 사칭 영구 정지 |
| 작가를 대접합니다 | 분양 경매, 인증 배지, 분양 이력 공개 |

## 2. 브랜드 성격 & 보이스

### 성격: "다정한 감정가(鑑定家)"
전문가의 눈 + 같은 덕후의 마음. 차갑지 않은 프로페셔널.

### 보이스 & 톤 규칙
1. **해요체.** "~하세요/~해요". 딱딱한 "~하십시오" 금지
2. **커뮤니티의 말을 쓴다.** 인형은 "아이", 소유자는 "집사", 작가 릴리즈는 "분양".
   단, 법적·금전 맥락(주문·결제·환불·제재)에서는 정확한 표준어로 전환
3. **돈 얘기는 정직하고 정확하게.** 수수료·기한·페널티를 숨기거나 얼버무리지 않는다.
   금액은 항상 원 단위 콤마 표기
4. **경고는 단호하게.** 팩토리 사칭·사기 관련 문구는 다정함을 걷어낸다 — "영구 이용 정지됩니다"
5. **버튼은 일어날 일을 그대로.** "입찰", "결제하기", "발송 등록" — 귀엽게 꾸미지 않는다

### 용어 사전
| 쓴다 | 안 쓴다 |
|---|---|
| 분양 (작가 릴리즈) | 판매, 드랍 |
| 아이 (인형, 감성 문맥) | 상품, 물건 (감성 문맥에서) |
| 집사 | 고객님 |
| 유찰 | 판매 실패 |
| 최저 낙찰가 | reserve price (화면 노출 시) |

## 3. 비주얼 아이덴티티

### 컨셉: "러버덕 옥션 하우스"
**덕션의 '덕'은 덕후의 덕이자 Duck.** 이름이 마스코트를 공짜로 준다 —
러버덕의 노랑은 친근하고, 옥션 하우스의 격식은 클린한 뉴트럴과 타이포 위계가 지킨다.
귀여움은 로고와 키 컬러 한 곳에만 허락하고, 나머지는 담백하게.

### 로고
- **마크: 러버덕** — 지오메트릭 러버덕 (원·타원 5개로 구성, `src/app/icon.svg` = 파비콘 겸용)
- **워드마크: "덕션" Pretendard ExtraBold** + 레터스페이스 영문 보조 "DUCTION · BLYTHE AUCTION"
- 조합: 마크 + 워드마크 (헤더·OG), 마크 단독 (파비콘·앱 아이콘)

### 컬러 (globals.css `@theme`과 1:1)
| 토큰 | 값 | 역할 |
|---|---|---|
| duck | `#FFD400` | **브랜드 키 — CTA 버튼 배경.** 옐로우 위 텍스트는 항상 ink |
| bill | `#C96A0E` | **기능 악센트 (부리색) — 가격·강조 텍스트·포커스.** 옐로우는 흰 바탕에서 글자가 안 읽히므로 텍스트 악센트는 bill이 맡는다 |
| cream | `#FDF6D8` | 소프트 하이라이트 배경 |
| porcelain | `#FCFCFA` | 그라운드 — 웜 화이트 |
| ink | `#26231C` | 텍스트 — 옐로우 기운의 웜 블랙 |
| mauve | `#8A857A` | 보조 텍스트 (웜 그레이) |
| blush / line | `#F6F4EE` / `#EAE8E1` | 서피스 틴트 / 경계선 |
| wisteria | `#6D5BA8` | 역할 컬러: 작가·분양 |
| verdigris | `#2E7D74` | 역할 컬러: 인증 |
| ok / warn | `#1F7A5C` / `#A16207` | 시맨틱 (악센트와 분리) |

### 타이포그래피
- **Pretendard 단일 패밀리** — 위계는 서체가 아니라 웨이트·크기로만 (깔끔함이 원칙)
  - 로고·페이지 제목: ExtraBold/Bold · 섹션 제목: SemiBold · 본문: Regular/Medium
- 숫자가 정렬되는 곳(가격·카운트다운·시세표)은 `tabular-nums` (`.num` 유틸리티)
- 우퍼케이스 라벨(eyebrow)은 `tracking-[0.25em]`

### 금지 사항
- 그라디언트 히어로, 이모지 섹션 마커, 세리프 혼용
- duck yellow에 흰 글자 올리기 (안 읽힌다 — 항상 ink)
- duck yellow를 넓은 배경면에 쓰기 (버튼과 로고의 색이지 벽지가 아니다)
- 시맨틱 컬러(ok/warn)를 장식으로 쓰기
